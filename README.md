## What is this?

This project is a C# TypeScript Definition Parser.

Features:

- Import and parse TypeScript Definitions to a light Abstract Syntax Tree
- Supports:
  - Classes
  - Comments
  - Imports
  - Interfaces
  - Methods
  - Properties
  - Types
